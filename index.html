Bueno, les dije que muy pronto haremos paginas webs, y
si se preguntan que es una pagina web, es un
conjunto de archivos que se guardan en un servidor y
se ejecutan por un navegador cuando alguien los llama por
su URL, estas se hacen con HTML y CSS (tambien
se usa JavaScript pero eso tu investigalo yo aqui no
vine a enseñarte a hacer webs dinamicas)
